7// Description: define a class for symmetric band matrix of double.
8// 注意事项：1. 矩阵的行号和列号从1开始，而不是从0开始
9// 2. 矩阵只保存下三角带内非零部分，按行存储，从对角元素开始，保存顺序如下

11// 1: [ | 1
12// 2: [ | 3 2
13// 3: [ | 6 5 4 DiagonalPos(k)

16// +------------+
17// Skyline(k)
18// ... [ 0 0 0 0 | * * ... *
19// ... [ ... ... ... ... ... ... ... ... ...
20// Notice: 1. the row and column number count from 1 not 0
21// 2. only the low triangle part in band of the matrix is saved in rows,
22// starting with diagonal elements as shown
23// 3. the band width is not constant and described by a Skyline
24// 1: [ | 1
